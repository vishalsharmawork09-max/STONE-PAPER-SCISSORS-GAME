✊📄✂️ Stone Paper Scissor Game
📌 Description

This is a simple Stone Paper Scissor game built using Python. The user plays against the computer, and the winner is decided based on standard game rules.

⚙️ Features
🎮 Player vs Computer gameplay
🎲 Random computer choice using random module
🧠 Rule-based winner decision
🔁 Menu-driven system (play again option)
📊 Simple and interactive output
🛠️ Technologies Used
Python
random module
basic conditional logic
▶️ How to Run
Install Python on your system
Download or clone this repository
Open terminal in project folder
Run the program:
python main.py
🎮 How to Play
Choose one option:
Stone ✊
Paper 📄
Scissor ✂️
Computer will randomly choose its option
Winner will be displayed after each round
You can play multiple rounds
